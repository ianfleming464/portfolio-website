To-Do App

Objective

My goal was to familiarise myself with Vue.js by building a small, fully-functional and responsive to-do app.

TRY ME OUT

Description

My Vue To-Do app is a single-page application which allows users to create to-dos, strike them through when complete, delete them and also edit them if needs be. The todos are persisted through the use of local storage. The app is styled with Vue Material.
