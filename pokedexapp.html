Pokemon Pokedex App

Objective

To build a small web application with HTML, CSS, and JavaScript that loads data from an external API and enables the viewing of data points in detail.

TRY ME OUT
View external API used

Description

This is a fully-responsive application which loads and presents info from the Pokemon Pokedex API, with full search functionality. Building this app was a great introduction to JQuery and Bootstrap, as well as a useful primer on ensuring applications are user-friendly, responsive and fully accessible.
